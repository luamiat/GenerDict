# Generador de contraseñas orientada a contraseñas por defecto

Este es un script hecho en python creado con el propósito de
crear diccionarios.

# ¿Comó ejecutarlo?
Para ejecutar GenerDict necesita tener python en su última versión
disponible si no lo tiene instalelo, después que lo instale
entre a la carpeta de GenerDict y ahí encontrara los 2 archivos
principales Gener.py y main.py para ejecutar el script
hagalo con el siguiente comando python main.py, procure estar
en la misma carpeta que los archivos antes de ejecutar el comando.
